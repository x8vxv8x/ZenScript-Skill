# CrTI18n（国际化）CraftTweaker API 参考

> Mod ID: 无
> 前置条件: ZenUtils
> 导入: `import mods.zenutils.I18n;`

国际化工具。

---

## API 列表

### I18n（国际化）

> `import mods.zenutils.I18n;`

#### 方法

| 方法 | 返回 | 说明 |
|------|------|------|
| `I18n.hasKey(String)` | bool | 检查翻译键是否存在 |
| `I18n.format(String)` | String | 根据翻译键获取本地化文本（等同于 `game.localize`） |
| `I18n.format(String, Object...)` | String | 翻译并格式化（等同于 `String.format(translate(key), args)`） |

---

## 使用示例

```zenscript
import mods.zenutils.I18n;

print(I18n.hasKey("item.apple.name"));         // true
print(I18n.format("item.apple.name"));         // "Apple"
```
