# CTGUI

CTGUI 允许你在游戏内通过 GUI 界面编辑配方。它会在本地脚本目录生成 `recipes.zs` 文件。

## 使用方法

在游戏内使用以下命令打开 GUI：

```
/CTGUI <id>
```

## 可用的 GUI ID

| ID | 说明 |
|------|------|
| `craftingtable` | 工作台配方编辑器 |
| `furnace` | 熔炉配方编辑器 |

## 注意事项

- 此命令**仅在单人模式**有效，在服务器上会返回 "Unknown Command"
- 如需在服务器使用，请在本地编辑后将生成的脚本文件上传到服务器
- 生成的文件位于本地脚本目录中的 `recipes.zs`
